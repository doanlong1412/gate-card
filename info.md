# 🚧 Gate Card

Smart gate control card for Home Assistant with live camera, animated SVG gate diagram, motion/person sensors, and a full visual editor.

## Features
- 🔄 Flip card — status face & control face
- 📷 Live camera snapshot (16:9)
- 🎨 SVG gate diagram with real-time position
- 🌐 10 languages with real flag images
- 🎨 16 background gradient presets
- 🎛️ Full visual config editor
